# Dotfiles

## Installation

Run:

    git clone https://github.com/vimist/dotfiles.git ~/.dotfiles
    ~/.dotfiles/setup.sh

## Palette

The basic 16 terminal colours that are recommended for se with these dot files are:

     1: rgb( 10,  10,  10)
     2: rgb(207, 106,  76)
     3: rgb(153, 173, 106)
     4: rgb(216, 173,  76)
     5: rgb( 89, 123, 197)
     6: rgb(160,  55, 176)
     7: rgb(113, 185, 248)
     8: rgb(173, 173, 173)

     9: rgb( 10,  10,  10)
